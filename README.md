# 企业微信H5静态收款码页面

## 使用说明

### 1. 准备收款码图片
- 将你的企业微信收款码保存为图片
- 命名为 `qrcode.png`
- 放入 `images` 文件夹

### 2. 准备企业Logo（可选）
- 将企业Logo保存为图片
- 命名为 `logo.png`
- 放入 `images` 文件夹

### 3. 修改企业信息
编辑 `index.html` 文件，修改以下内容：
- 企业名称
- 联系电话
- 版权信息

### 4. 部署方式

#### 方式一：直接使用（本地）
直接用浏览器打开 `index.html` 即可预览

#### 方式二：部署到服务器
将所有文件上传到你的Web服务器，通过域名访问

#### 方式三：使用GitHub Pages（免费）
1. 创建GitHub仓库
2. 上传所有文件
3. 在仓库设置中启用GitHub Pages
4. 获得访问链接

#### 方式四：使用Gitee Pages（国内访问快）
1. 创建Gitee仓库
2. 上传所有文件
3. 在服务中启用Gitee Pages
4. 获得访问链接

### 5. 在企业微信中使用
- 将页面链接发送到企业微信
- 客户点击链接即可查看收款码
- 支持转发给其他联系人

## 文件结构
```
H5_pay/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 脚本文件
├── README.md           # 说明文档
└── images/             # 图片文件夹
    ├── logo.png        # 企业Logo
    └── qrcode.png      # 收款码
```

## 特点
- ✅ 纯静态页面，无需后端
- ✅ 响应式设计，适配各种手机
- ✅ 支持长按识别二维码
- ✅ 美观的渐变背景
- ✅ 可自定义企业信息
- ✅ 轻量级，加载快速

## 自定义建议
1. 修改 `style.css` 中的渐变色以匹配企业VI
2. 添加更多支付方式说明
3. 可以添加收款金额输入功能（需要动态二维码）
4. 添加统计代码追踪访问量

## 注意事项
- 确保收款码图片清晰可识别
- 定期检查收款码是否过期
- 建议使用HTTPS协议部署
- 可以添加访问密码保护（需要后端支持）
